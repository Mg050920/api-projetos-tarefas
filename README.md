# API CRUD em Dart

Este projeto é uma API simples desenvolvida em Dart para praticar os conceitos de CRUD: criar, listar, atualizar e remover registros.

## Tecnologias utilizadas

- Dart
- Servidor HTTP
- JSON

## Funcionalidades

- Criar um registro
- Listar registros
- Atualizar um registro
- Remover um registro
